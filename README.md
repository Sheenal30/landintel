# 🗺️ LandIntel: Transparent Land Intelligence for India

**Live Demo:** [Insert your GitHub Pages link here, e.g., https://yourusername.github.io/landintel-mvp/]

LandIntel is a geographic intelligence platform designed to bring transparency to the Indian real estate market, starting with Jaipur, Rajasthan. 

Currently, retail buyers in India fall victim to information asymmetry. Brokers know where future infrastructure (metros, ring roads, industrial zones) is being built, while middle-class buyers do not. LandIntel aggregates municipal infrastructure data and correlates it with local land listings to generate "Growth & Risk" assessments.

## 🚀 The MVP
This repository contains the Phase 1 MVP, built entirely with zero-budget, open-source tools by a self-taught developer. 
- **Tech Stack:** HTML5, CSS3, Vanilla JavaScript, Leaflet.js, OpenStreetMap.
- **Features:** 
  - Interactive map plotting approved/proposed infrastructure against land listings.
  - Price filtering algorithms.
  - Spatial risk/growth assessments based on proximity to development zones.
